Git - это распределенная система контроля версий. Предназначен для отслеживания и ведения истории изменения файлов, в вашем проекте

Github это сайт куда мы можем отправлять все что находится на нашем ноутбуке, чтобы другие люди могли могли посмотреть на мой проект и стянуть к себе

1. git init - что бы создать скрытую директорию (.git), инициализация
2. git add - просматривает изминения в нашем проект, добавляет в промежуточное пространство (index)
3. git commit -m 'massage' - фиксирует изменения из индекса в локальный репазиторий
4. git remote add {название связки(origin)} - связывает локальный и удаленный репазиторий
5. git push {название связки(origin) main либо master} - отправляет данные из локального в удаленный репазиторий
6. git pyll origin main - отправляет данные из удаленного в локальный репазиторий


Закидываем изменения в готовый репазиторий
1. git add git_github.txt(название файла)
2. git commit -m 'краткое сообщение на английском языке'
3. git origin master или main


Стягиваем изменения из удаленного репазитория
1. git clone и ссылка из git hub
2. git pull origin main
